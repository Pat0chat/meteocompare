## Politique des dons

- Les dons sont **anonymes par défaut**. Aucun privilège, accès premium,
  ou contenu exclusif n'est lié aux dons.
- Le code source reste **identique pour tous**. Pas de version "donateurs".
- Les dons financent le **temps consacré au développement** (corrections de
  bugs, nouvelles features, support de nouveaux modèles météo) et les
  éventuels coûts (compte développeur Play Store, hébergement de la
  politique de confidentialité, etc.).

## Conformité Play Store

Conformément aux règles de Google Play (section "Donations & charity"),
MeteoCompare :
- N'utilise pas Google Play Billing pour les dons (interdiction sur les
  apps non-charity)
- Ouvre simplement un navigateur web vers les plateformes externes
- Ne déclare aucun statut de charité ou tax-deductible

## Autres façons de contribuer

Le code, pas l'argent — c'est aussi très bienvenu :

- **Bug reports** : [github.com/USERNAME/MeteoCompare/issues](https://github.com/USERNAME/MeteoCompare/issues)
- **Pull requests** : voir `CONTRIBUTING.md` si présent, sinon ouvrir une
  discussion d'abord pour les changements significatifs
- **Traductions** : l'app est en français, des PRs pour ajouter d'autres
  langues sont les bienvenues (l'infra `fastlane/metadata` supporte déjà
  les locales multiples)
- **Beta testing** : rejoindre le track de test sur Play Console pour
  rapporter les régressions avant release
